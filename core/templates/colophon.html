This site was meticulously crafted using a variety of technologies and services to ensure its performance, security, and reliability. Below is a brief overview of the tools and platforms powering TechRahis.

1. Backend

The backend is built with Django, a high-level Python web framework that encourages rapid development and clean, pragmatic design. The data is stored securely in PostgreSQL, a powerful, open-source object-relational database system known for its robustness and scalability. For in-memory data storage and caching, I use Redis, which ensures fast access to frequently requested data.

2. Frontend & Hosting

The frontend is developed using modern web technologies and hosted on Vercel, a platform for frontend frameworks and static sites that handles the deployment with seamless integration and automatic scaling.

3. Security & Performance

To ensure site security, performance, and efficient global content delivery, I rely on Cloudflare. It provides DDoS protection, caching, and a global content delivery network (CDN) that reduces latency and enhances site performance for visitors worldwide.

4. Version Control & Continuous Integration

The source code for this website is maintained and versioned using Git and hosted on GitHub. Vercel integrates seamlessly with GitHub, enabling continuous deployment with each update.

5. Design & Styling

The design of this site is styled with Tailwind CSS, a utility-first CSS framework. It allows for rapid development and ensures a responsive, clean, and consistent design across devices.

I am grateful for these incredible open-source tools and services that help me bring this site to life and deliver a seamless user experience.
